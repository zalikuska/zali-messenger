Network

Сервер и WebRTC

public

TURN помогает звонкам работать вне локальной сети. Обычно нужен `turns://` адрес, логин и credential от coturn.

API и WebSocket лучше указывать публичными `https`/`wss` адресами. Для работы через интернет добавьте TURN-сервер в список ICE.
