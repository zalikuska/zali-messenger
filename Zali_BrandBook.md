# Zali Visual Identity & Brand Book v1.0

Добро пожаловать в визуальный мир **Zali**. Этот документ описывает философию дизайна "Cyber-Glass" и предоставляет конкретные параметры для реализации интерфейсов в стиле Zali.

---

## 1. Философия: Cyber-Glass
Дизайн Zali базируется на трех столпах:
- **Глубина**: Использование слоев, теней и размытия (Backdrop Filter).
- **Контраст**: Сочетание абсолютной тьмы и неонового света.
- **Минимализм**: Ничего лишнего, только функциональные элементы.

---

## 2. Цветовая палитра

| Название | HEX | Применение |
| :--- | :--- | :--- |
| **Zali Lime** | `#cbff00` | Акценты, кнопки, важные статусы, свечение. |
| **Deep Dark** | `#090a0c` | Основной фон приложения. |
| **Glass Base** | `rgba(255,255,255,0.03)` | Фон контейнеров и карточек. |
| **Glass Border** | `rgba(255,255,255,0.08)` | Границы элементов (1px). |
| **Text Primary** | `#ffffff` | Основные заголовки и текст. |
| **Text Secondary** | `rgba(255,255,255,0.6)` | Описания, подписи, неактивные элементы. |

---

## 3. Типографика
**Основной шрифт:** `Inter` или `Segoe UI` (System Sans).
- **Headers (H1, H2):** Bold (900), Letter-spacing: 2px, All-caps для логотипов.
- **Body:** Medium (500), Line-height: 1.5.
- **Monospace:** Для путей к файлам и хэшей.

---

## 4. Эффекты Glassmorphism (CSS)
Чтобы добиться эффекта Zali, используйте следующие параметры для контейнеров:

```css
.zali-card {
    background: rgba(255, 255, 255, 0.03);
    backdrop-filter: blur(40px);
    -webkit-backdrop-filter: blur(40px);
    border: 1px solid rgba(255, 255, 255, 0.08);
    border-radius: 16px;
    box-shadow: 0 20px 50px rgba(0, 0, 0, 0.6);
}
```

---

## 5. Компоненты

### Кнопки (Primary)
- **Фон:** `#cbff00`
- **Текст:** Black (`#000000`)
- **Тень:** `0 0 15px rgba(203, 255, 0, 0.3)`
- **Hover:** Увеличение яркости и легкий подъем (translateY).

### Карточки файлов
- **Стиль:** Плитка (Grid).
- **Иконка:** Упрощенная, в цвете Zali Lime.
- **Индикаторы:** Полоса прогресса или точка статуса в цвете Lime.

## 7. Голос бренда (Tone of Voice)
Zali говорит на языке **уверенности и технологий**. 
- Избегайте лишних слов.
- Используйте четкие глаголы (Упаковать, Извлечь, Защитить).
- Подчеркивайте безопасность (AES-256, RSA).
